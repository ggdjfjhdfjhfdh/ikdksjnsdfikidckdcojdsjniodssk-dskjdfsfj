Archivo de verificación para Google Search Console

Para configurar correctamente:

  1. Accede a Google Search Console
  2. Agrega tu propiedad (https://sesecpro.es)
  3. Selecciona el método de verificación "Archivo HTML"
  4. Descarga el archivo y reemplaza este contenido
  5. Sube el archivo a la raíz del sitio web
